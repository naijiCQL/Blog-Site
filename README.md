# Blog Site

## 项目简介

1、本项目目前采用Node.js + Express + EJS进行编写，数据库采用MongoDB，主要目的是学习Node.js。

2、部分代码未进行展示，目前项目正在持续更新中。

3、欢迎大家一起沟通交流。

## 项目启动

### 依赖安装

> npm install

### 启动项目

> npm run start
